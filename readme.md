## Bilibili-Downloader

Homework of IS305

## Requirements

- Python3
- PyQt5
- FFmpeg
- Potplayer(to play mp4 offline)

## Run the code

change to the code directory
```sh
pip install -r requirements.txt
python gui.py
```

## To pack

modify the path of ffmpeg and potplayer
```sh
pyinstaller -Fw gui.py
```
run the exe in the previous directory
